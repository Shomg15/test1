# 一般操作熱鍵

- <kbd>Ctrl +</kbd> : 將畫面放大
- <kbd>Ctrl -</kbd> : 將畫面縮小
- <kbd>Ctrl f</kbd> : 檔案搜尋
- <kbd>Ctrl Shift f</kbd> : 工作區搜尋
- <kbd>Ctrl `</kbd> : 顯示終端機(輸入指令的地方)
- <kbd>Ctrl F2</kbd> : 將相同的部分全部選取
- <kbd>Ctrl d</kbd> : 將相同的部分逐一選取

## 設定操作

- <kbd>ctrl shift p</kbd> : 輸入 `OpenUser Settings (JSON)`, 可以開啟使用者設定
